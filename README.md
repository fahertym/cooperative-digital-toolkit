
# Cooperative Digital Toolkit

A member-owned, open-source platform for cooperative governance, finance, and collaboration.

## Quick Start (Dev)
- Backend: Go
- Frontend: Svelte
- DB: PostgreSQL
- Automations: n8n; Imports: Airbyte

```bash
make dev   # run backend + frontend in watch mode
make test  # run tests
```
